# Kingkan's Portfolio Website

A modern, high-performance personal portfolio showcasing expertise in Mobile Engineering, Architecture, and Security. Built with a focus on polished UI, responsive design, and enterprise-grade deployment.

## 🚀 Tech Stack

### Frontend
- **Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Custom CSS with CSS Variables (Vanilla approach for maximum performance and pixel-perfect control)
- **Animations:** Framer Motion (for smooth micro-interactions and modal transitions)
- **Icons:** `react-icons` / `lucide-react`

### Infrastructure & Deployment
- **Containerization:** Docker (Multi-stage build using Node.js builder + Nginx edge server)
- **Hosting:** Google Cloud Run (Serverless, Auto-scaling container platform)
- **CI/CD:** GitHub Actions / Cloud Build integrations
- **DNS/CDN:** Cloudflare (Custom Domain + Proxy)

## 📦 Project Setup

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
